# SIMPLEX-ITY MONITORING MASTER PLAN
# All surfaces — Brands, Influencers, Campaigns, Platform, Tools
# Built by Family — June 7, 2026

## WHAT WE ARE TRACKING

### BRAND MONITORING (per brand, monthly)
- GMV generated this month
- Number of products live on platform
- Number of campaigns active
- Brand satisfaction score (1-10)
- Fulfilment time (days from order to ship)
- Return/refund rate (%)
- Tier status + renewal date
- Last contact date (Loreen)
- Next action required

### INFLUENCER MONITORING (per influencer, per campaign)
- Lives agreed vs lives done
- Blog posts agreed vs done
- Total GMV driven
- Commission earned
- Reliability score (auto-calculated)
- Audience engagement rate
- Best performing product
- Best performing time slot
- AI certified status
- Next live date

### CAMPAIGN MONITORING (per campaign)
- Brand + influencer assigned
- Start and end date
- Lives done vs target
- GMV generated
- 15% commission captured
- Status (active/completed/paused)
- Issues flagged

### PLATFORM HEALTH (weekly)
- Tier 1-4 checks (16 surfaces)
- Wilson deliverables status
- Tool subscription status

### TOOL SUBSCRIPTION TRACKER
- Tool name
- Cost per month
- Free trial start date
- Free trial end date
- Paid start date
- What it monitors
- Status (free/paid/not started)
